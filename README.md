# CV Job Role Predictor

This project predicts IT job roles from CV/resume data using various machine learning models. It includes data exploration, preprocessing, feature engineering, model training, evaluation, and deployment with FastAPI.

## Project Structure
- **data/**: Dataset(s) for training and evaluation
- **notebooks/**: Jupyter notebooks for each step of the ML workflow
- **models/**: Saved models
- **deployment/**: FastAPI app and deployment files

## Usage
1. Explore and preprocess data in the notebooks.
2. Train and evaluate models.
3. Deploy the best model using FastAPI in the deployment folder.
